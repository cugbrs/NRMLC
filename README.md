# NRMLC-A-Noise-Robust-Multilabel-Remote-Sensing-Scene-Classification-Framework

Multilabel Remote Sensing Scene Classification (ML-RSSC) based on deep learning (DL) models aims to assign multiple labels to a single remote sensing (RS) image, which has been widely utilized in object detection, resource investigation, and environmental protection. Label noise, coupled with object scale variation and class imbalance, significantly degrades the performance of ML-RSSC. To address these challenges, this paper proposed a Noise-Robust Multi-Label Classification (NRMLC) framework. The architecture integrated three core components: (1) a Multi-Scale Feature Extraction (MSFE) module to capture multi-granularity features for objects of varying sizes; (2) a Label-Wise Enhanced Attention (LWEA) module to mitigate label imbalance by enhancing class-specific representations; and (3) a Transformer-Based Multi-Label Classification (TBMC) module to correct noisy signals via semantic dependency modelling.
